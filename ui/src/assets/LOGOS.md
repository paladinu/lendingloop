# Lending Loop Logos

This directory contains various logo designs for the Lending Loop application.

## Logo Files

### 1. logo-icon.svg (Currently Used in Toolbar)
- **Size**: 48x48px
- **Style**: Compact icon with interlocking loops forming an infinity symbol
- **Use Case**: Toolbar, favicon, app icons
- **Features**: Clean, minimal design perfect for small spaces

### 2. logo-modern.svg
- **Size**: 240x60px
- **Style**: Horizontal layout with interlocking circles and text
- **Use Case**: Headers, landing pages, marketing materials
- **Features**: Full branding with gradient effect

### 3. logo-circular.svg
- **Size**: 200x200px
- **Style**: Circular badge with infinity loops and "L" monogram
- **Use Case**: App icons, social media profiles, badges
- **Features**: Bold circular design with gradient background

### 4. logo-minimal.svg
- **Size**: 200x60px
- **Style**: Stylized "LL" with loops and text
- **Use Case**: Email signatures, documents, alternative header
- **Features**: Elegant minimal design with double-L motif

## Design Concept

All logos feature the concept of **interlocking loops** representing:
- Connection and community
- The lending/borrowing cycle
- Continuous sharing and trust
- Infinity symbol suggesting endless possibilities

## Color Scheme

Primary colors match the Angular Material theme:
- Primary Blue: #3f51b5
- Light Blue: #5c6bc0
- Accent Blue: #7986cb
- White for contrast

## Usage

The toolbar currently uses `logo-icon.svg` with a white filter applied for visibility on the primary-colored toolbar. Other logos can be used throughout the application as needed.
